#### Installation

###### Via Composer
add a following line (root-only) into your composer.json

    "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/TerranetMD/zf-google-translate.git"
        }
    ]
    
run
    
    composer update

###### Via GitHub

    git clone https://github.com/TerranetMD/zf-google-translate.git
